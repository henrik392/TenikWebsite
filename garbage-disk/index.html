Garbage Disk

About

Sort all the files

Garbage disk is an endless score based game. The goal is to sort all the files that comes into the computer. You only have space for 20 files, so be quick so you don't run out of space! This game was made in Ldjam for the theme running out of space. Try beat my personal best score at 48.
